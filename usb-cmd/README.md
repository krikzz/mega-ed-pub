# USB Command Examples

| Path | Description |
|---|---|
| `rst_hard.py` | Full system reset using cartridge pin `B2`. Equivalent to a cold boot. |
| `rst_soft.py` | System reset using cartridge pin `B27`. Equivalent to pressing the console reset button. |
| `run_app_fpga_core.py` | Runs `ssf-tst.md` using a custom mapper on Mega EverDrive CORE. |
| `run_app_fpga_pro.py` | Runs `ssf-tst.md` using a custom mapper on Mega EverDrive PRO. |
| `run_app.py` | Runs `ssf-tst.md`. |
| `ssf-tst.md` | Test ROM using the SSF mapper. |
